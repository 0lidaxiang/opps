Setup
-----

Installation
************

You can use pip to install Opps and requirements::

    pip install opps

or::

    $ git clone git@github.com:opps/opps.git
    $ cd opps
    $ python setup.py install


Start project
*************

You can use `opps-admin.py` to start new project::

    $ opps-admin.py startproject PROJECT_NAME
